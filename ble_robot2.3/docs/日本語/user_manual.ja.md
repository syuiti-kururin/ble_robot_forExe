# ユーザーマニュアル (日常運用)

この文書は、配布済みアプリを毎日使う操作者向けです。開発作業ではなく、運用の安定性と再現性を重視した使い方をまとめています。

## 基本操作フロー

1. ble_robot_2.3.exe を起動
2. 必要に応じて接続先を確認
3. Connect で接続
4. 手動操作または自動モードを利用
5. 通常終了で設定保存

## 保存の仕組み

ユーザー操作で変更した値は cf.rbcc に保存されます。内部管理値は meta/*.rbci に保存されます。About/License も rbci 側を参照するため、運用中の文面差し替えが可能です。

## 変更時の注意

- 手編集時はバックアップを作成する
- 複数端末配布時は cf.rbcc と meta のバージョンをそろえる
- 設定変更後は再起動して反映確認を行う

## よくある問題

接続失敗時は、BLE アドレス・ロボット状態・PC 側 Bluetooth 状態を順に確認してください。設定が戻る場合は、書き込み権限やセキュリティソフトの隔離を確認してください。表示文面が更新されない場合は、対象 rbci を更新したかを確認し、画面を開き直してください。
