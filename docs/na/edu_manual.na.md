# EDU Manual (For Class Instructors)

In educational use, minimizing failure in session flow is more important than explaining every feature first. This document provides a practical class template.

## Principles for Lesson Design

- Perform safety checks first
- Keep one objective per session
- Create early success experience
- Explain principles at the end

## Recommended Flow (45-90 min)

1. Share objective: what students will achieve today
2. Demonstration: from connect to one movement
3. Practice: students execute the same steps
4. Adjustment: troubleshoot connection failures together
5. Reflection: verify persistence and reproducibility

## How to Teach File Roles

Explain cf.rbcc to learners as their own settings file.

## Reproducibility After Class

Keep backups of cf.rbcc and meta after class and run diff checks before the next class. This improves stability in multi-class operation.
