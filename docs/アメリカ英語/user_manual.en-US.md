# User Manual (Beginner / EXE-only)

## Scope
- For users who only run the distributed EXE.

## Quick Start
1. Launch `ble_robot_2.2.exe`.
2. Open settings and set BLE address.
3. Click `Connect`.
4. Drive with buttons/gamepad.
5. Exit normally to save `cf.rbcc`.

## Notes
- If connection fails, verify BLE address and robot power.
