# Starter Manual (First-time ROBO-ONE Beginner)

## Goal
- First setup for robot + PC control app.

## Thonny setup
1. Install Thonny.
2. Choose interpreter/device profile.
3. Verify serial connection.
4. Deploy robot-side script.
5. Run basic send/receive test.

Then launch the EXE and connect over BLE.
