# Manual de Administración (Alcance Público)

Documento para responsables de distribución, publicación y gobierno operativo. Describe solo información publicable.

## Separación de responsabilidades

- cf.rbcc: fuente de verdad de ajustes de usuario
- meta/*.rbci: datos operativos internos
- meta/*.rbcl: datos de logotipo y relacionados

La separación existe para mantener cambios post-distribución de forma segura y evitar editar directamente el ejecutable.

## Estructura recomendada

- ble_robot_2.3.exe
- cf.rbcc
- meta/
- tools/
- docs/

## Política de cambios

- Ajustes de experiencia de usuario en cf.rbcc
- Ajustes operativos/internos en rbci
- Historial y respaldo por unidad de distribución

## Lista de verificación

1. Inicio y conexión
2. Persistencia de ajustes
3. Reflejo de cambios rbci
4. Inicio de tools
5. Consistencia documental
