# Manual de Usuario (Principiante / solo EXE)

## Alcance
- Para uso solo con el ejecutable distribuido.

## Inicio rápido
1. Ejecuta `ble_robot_2.2.exe`.
2. Configura dirección BLE en ajustes.
3. Presiona `Connect`.
4. Controla con botones o gamepad.
5. Cierra normalmente para guardar `cf.rbcc`.

## Nota
- Si falla la conexión, revisa dirección BLE y energía del robot.
