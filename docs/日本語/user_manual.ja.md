# ユーズマニュアル (初心者向け / exe利用のみ)

## 対象
- Python環境を使わず、配布されたexeだけで操作したい人。

## 使い方
1. `ble_robot_2.2.exe` を起動。
2. 歯車ボタンで BLE アドレスを設定。
3. `Connect` を押して接続。
4. 十字キーまたはゲームパッドで操作。
5. 終了時は通常終了（設定は `cf.rbcc` に保存）。

## トラブル時
- 接続失敗: BLEアドレスと電源状態を確認。
- 設定が保存されない: `RBCC file` のログを確認。
