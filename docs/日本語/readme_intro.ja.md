# README (導入)

ROBO BLE Controller 2.3 は、ROBO-ONE Beginner の学習と実演に使う Windows 向け操作アプリです。BLE 接続でロボットを制御し、受信データの可視化、設定の保存、授業現場での再現運用を行えるように設計されています。

この配布では、利用者が編集する設定と、運用者が管理する内部設定を分離しています。これにより、配布後の設定変更を安全に行いながら、実行ファイル本体を再ビルドせずに運用を継続できます。

## 現行構成 (公開版)

- 実行ファイル: ble_robot_2.3.exe
- ユーザー設定: cf.rbcc
- 内部設定: meta/*.rbci
- ロゴ関連: meta/*.rbcl
- ドキュメント: docs/配下

## 設定分離の意図

cf.rbcc は操作者が日常的に変更する項目を保持します。meta/*.rbci は配布後に調整が必要な内部情報を保持します。内部情報を別ファイル化することで、exe 本体に固定値を埋め込む運用を避け、保守性とデータ保護を両立します。
